## Projetos do Trens
Simulação de uma malha ferroviária composta de 9 regiões críticas, com o objeto de coordenar os trens usando `mutex` e semáforos.

Cada trem executa em `thread` indenpendente da outra, sem impasses, travamentos ou _deadlocks_.

### Como executar
* Basta abrir o projeto (`Trem.pro`) no Qt Creator e clicar em __build__.
